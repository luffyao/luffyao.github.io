Ubuntu 18.04 上安装 Docker CE

Posted by Luffyao on Sunday, December 1, 2019

安装

官方网站上选择一个合适的安装方法。请参考 官方文档。

注意

Maybe you will appear docker-ce-cli not found issue,when using following command which written by docker formal docs to install docker ce. you can refernce my solution to solve it. 使用 Docker 官方文档编写的以下命令安装 Docker CE 时，可能会出现 docker-ce-cli 未发现问题。你可以参考我的方法来解决它。

从下面的命令中删除 “docker-ce-cli=<VERSION_STRING>" 部分。也可以参考 ubuntu-for-docker-ce-cli-was-not-found。

sudo apt-get install docker-ce=<VERSION_STRING>docker-ce-cli=<VERSION_STRING> containerd.io

为 Docker 命令添加权限

sudo groupadd docker;sudo usermod -aG docker $USER
